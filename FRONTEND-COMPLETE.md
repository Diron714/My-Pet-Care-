## 📦 Next Steps

1. **Install Dependencies:**
   ```bash
   cd frontend
   npm install
   ```

2. **Create .env file:**
   ```env
   VITE_API_BASE_URL=http://localhost:5000/api
   ```

3. **Start Development Server:**
   ```bash
   npm run dev
   ```

4. **Build for Production:**
   ```bash
   npm run build
   ```
